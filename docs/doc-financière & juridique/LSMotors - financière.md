# **Conclusion**

Le projet **LS Motors** représente un investissement simulé de **12 696 € TTC** pour un développement complet, fonctionnel et documenté.  
Cet investissement est justifié par :

- l’amélioration de l’efficacité interne,  
- la professionnalisation du système,  
- le réalisme RP apporté à la concession,  
- une maintenance annuelle maîtrisée,  
- une architecture évolutive.

La solution développée par **Nyght** offre un excellent rapport qualité/prix et peut évoluer selon les futurs besoins de LS Motors.
