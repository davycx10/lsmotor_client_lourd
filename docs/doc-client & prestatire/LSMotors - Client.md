# **Contexte**

Le projet LS Motors s’inscrit dans l’univers **GTA V**, **FiveM** et du **Roleplay (RP)**.

- GTA V est un jeu en monde ouvert développé par Rockstar Games.  
- FiveM est un mod multijoueur permettant la création de serveurs personnalisés.  
- Le RP consiste à incarner un personnage dans une ville virtuelle (Los Santos) et à simuler des activités réalistes.

Sur le serveur **District RP**, LS Motors est une **concession automobile haut de gamme**, avec :

- des employés,  
- un catalogue de véhicules,  
- des prix,  
- des ventes,  
- une organisation interne.

Pour professionnaliser cette activité, l’entreprise **NYT** a été sollicitée afin de développer une **plateforme web complète**, utilisée à la fois :

- en jeu (RP),  
- en gestion interne,  
- en consultation publique.

---

# **LS Motors – Présentation générale**

LS Motors est une concession automobile premium située à Los Santos (serveur District RP).

La plateforme web développée par NYT permet de :

- gérer les véhicules,  
- gérer les employés,  
- gérer les catégories & marques,  
- organiser les ventes,  
- afficher publiquement l’inventaire,  
- améliorer la productivité et la transparence RP.

L’objectif : **refléter l’univers premium de LS Motors** à travers un outil moderne, efficace et simple d’utilisation.

# **Activités et services de la concession**

## Vente de véhicules

Catégories RP disponibles :

- Sport  
- Super  
- SUV  
- Compacts  
- Motos  
- Classics  
- Off-road  
- Utilitaires  
- Luxe  
- Etc.

Chaque véhicule inclut :

- prix catalogue,  
- prix après taxes RP,  
- image illustrative,  
- informations techniques (optionnel).  

## Accompagnement client

Les employés conseillent les joueurs sur :

- le choix du véhicule,  
- la disponibilité,  
- les formalités d’achat RP.

##  Gestion interne

La concession gère :

- employés,  
- ventes,  
- véhicules,  
- marques,  
- catégories.

Ces besoins ont motivé la création d’un **outil web automatisé**.

# **Parcours client & gestion interne**

## 🔸 Avant la visite
Le joueur consulte sur le site :

- catalogue,  
- catégories,  
- prix,  
- images,  
- disponibilités.

## 🔸 Sur place (RP)
Le vendeur utilise la version admin pour :

- vérifier les stocks,  
- afficher les véhicules par catégorie,  
- afficher les prix RP.

## 🔸 Lors de l’achat
L’employé peut enregistrer la vente dans le système LS Motors.

## 🔸 Avantages du site
- Gain de temps  
- Professionnalisation du RP  
- Fiches véhicules complètes  
- Outil universel pour tous les employés

# **Organisation et fonctionnement**

Deux rôles principaux :

## **Employé**
- Consultation du catalogue  
- Accès aux catégories / marques  
- Affichage des véhicules  

## **Administrateur**
- Ajouter / modifier / supprimer un véhicule  
- Gérer les utilisateurs  
- Gérer catégories et marques  
- Configurer le site  

Toutes ces fonctionnalités sont implémentées via une **architecture MVC sécurisée**.
