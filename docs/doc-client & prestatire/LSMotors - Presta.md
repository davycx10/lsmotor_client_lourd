# **Contexte de création de l’entreprise**

L’entreprise **NYT (Night)** est née de la volonté de quatre étudiants en BTS SIO SLAM de créer une structure fictive capable de répondre à des besoins professionnels réels en développement web.

NYT simule une entreprise moderne spécialisée dans :

- la conception d’applications web sur mesure,  
- des outils métiers performants,  
- des interfaces ergonomiques,  
- des systèmes sécurisés (MVC, rôles, BDD).  

L’objectif est de reproduire le fonctionnement d’une véritable société de développement :

- analyse du besoin,  
- conception,  
- développement,  
- tests,  
- déploiement,  
- maintenance.  

Le projet **LS Motors** est l’un des projets majeurs de NYT, démontrant :

- une méthodologie professionnelle,  
- une maîtrise technique,  
- une réponse adaptée à un besoin métier RP.

# **Enjeux dans le développement du projet LS Motors**

Le projet LS Motors représente un enjeu stratégique pour NYT.  
Voici les objectifs majeurs :

---

## 🔸 **1. Répondre à un besoin métier clair**

Le client souhaitait un outil permettant de :

- gérer un catalogue de véhicules RP,  
- gérer employés, marques, catégories, ventes,  
- afficher publiquement les véhicules,  
- sécuriser l’accès via rôles.

**Enjeu :** créer une plateforme claire, performante et intuitive.

---

## 🔸 **2. Mettre en avant les compétences techniques de NYT**

Le projet démontre :

- architecture MVC structurée,  
- sécurité (rôles, sessions, PDO),  
- interface admin complète,  
- design moderne cohérent avec LS Motors.

---

## 🔸 **3. Démontrer la capacité à gérer un projet complet**

Le projet a nécessité :

- analyse fonctionnelle,  
- réflexion UX,  
- conception BDD,  
- développement PHP/MySQL,  
- intégration front-end,  
- tests,  
- documentation.

**Enjeu :** prouver qu’une équipe de 3 développeurs peut mener un projet complet.

---

## 🔸 **4. Renforcer la relation client dans un univers RP**

Même en RP, les enjeux restent professionnels :

- fluidifier les ventes RP,  
- renforcer l’identité de la concession,  
- structurer le travail des employés,  
- améliorer la confiance client ↔ prestataire.

---

## 🔸 **5. Offrir une expérience utilisateur moderne et immersive**

NYT a conçu une interface :

- simple,  
- responsive,  
- cohérente (charte doré/noir),  
- adaptée au public RP,  
- rapide à utiliser en scène RP.

**Enjeu :** proposer une expérience premium fidèle à LS Motors.
