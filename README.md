# lsmotor_client_lourd
client lourd ppe

**Prérequis** :
Java *11+*
MySQL installé et lancé en local
Connecteur JDBC MySQL dans les dépendances (mysql-connector-java) 
Dans IntelliJ <-- pour *mysql:mysql-connector-java* : **File** → *Project Structure* → **Libraries Clique sur + → From Maven** Tape : *mysql:mysql-connector-java:8.0.33* (ou version superieur)
La base créée avec le script SQL fourni (dans lsmotor_admin)
